## 🎯 The Problem It Solves

> *"What should I eat today?"* — asked by everyone, every single day.

**MealPickup** instantly solves food decision fatigue by randomly suggesting meals — saving you time and mental energy!

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎲 **Random Picker** | Instantly suggests a random food option |
| 🍔 **Multiple Options** | Wide variety of food choices available |
| 🔄 **Regenerate** | Don't like it? Get a new suggestion instantly |
| ⚡ **Fast & Interactive** | Smooth React-powered dynamic UI |
| 📱 **Fully Responsive** | Works on mobile, tablet & desktop |
| 🎨 **Clean UI** | Minimal and user-friendly design |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/sakshimalkar/MealPickup.git

# Navigate into the project
cd MealPickup

# Install dependencies
npm install

# Start development server
npm start
```
