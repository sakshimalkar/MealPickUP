## 🍽️ What To Eat Picker

A modern and responsive web application that helps users quickly decide what to eat when they are confused about food choices.

The application randomly selects food items from a list and suggests them to the user, making food decisions easier and faster.

This project is built using React.js for dynamic UI components and Bootstrap for a responsive and clean interface.

🌐 Live Demo

🔗  https://sakshimalkar.github.io/MealPickUP/

🚀 Features
🎲 Random food suggestion generator
🍔 Different food options to choose from

🔄 Generate a new suggestion instantly
